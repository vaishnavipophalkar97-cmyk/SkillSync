# SkillSync
🤖 An AI-powered resume analyzer that parses PDFs, evaluates skills against job descriptions, and provides tailored optimization feedback using LLMs.
It is designed to help HR teams and recruiters efficiently filter and evaluate candidates based on Job Description .

The system allows users to upload multiple resumes along with a job description. Using Gemini AI integration , the platform analyzes each resume and provides detailed insights such as:
1.Candidate suitability
2.Strengths and Weakness
3.Skill Gaps
4.Matching Percentage 
5.Recommendation status

## Features
### Multiple Resume Uploads
Upload and analyze multiple resumes simultaneously
### Gemini Integration
Uses Gemini API for intelligent resume understanding and candidate evaluation.
### Detailed Skill Analysis
For every candidate , the system generates:
#### Skill Match Score
#### Key Strengths
#### Missing Skills
#### Experience Relevance
#### Final Recommendation

## Workflow
1. Upload multiple resumes
2. Enter or upload the Job Description
3. Gemini AI analyzes resumes against JD
4. System generates detailed evalution
5. Candidates are categorized as :
   * Selected
   * Backup
   * Rejected
